One advantage of this template is that you can quickly make and view edits on your local computer. Go to your GitHub repo and select Code > Download Zip, uncompress the folder, and install a free code editor tool such as Atom Editor, and open the "index.html" file to make edits directly on your local computer. View your edits by opening "index.html" with any browser, which is faster than waiting for GitHub Pages to update your online version every time you make a small change. After making several edits on your local computer, carefully delete the old "index.html" file in your online GitHub repo, and replace it with the updated "index.html" file from your local computer.
